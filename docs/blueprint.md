# **App Name**: BizForge

## Core Features:

- Secure User Authentication & Role Management: Implement Firebase Authentication for multi-role access (Super Admin, Company Owner, Employee, etc.) ensuring secure tenant isolation.
- Global Admin Platform: Provide a centralized interface for Super Admins to manage registered companies, assign licenses, define pricing plans, and enable or disable specific modules for each company. Includes company listing and detail views.
- Company-Specific Portal & Dashboards: Each registered company receives a dedicated portal with a customized dashboard displaying relevant overview cards and tables based on activated modules.
- Employee & Customer Management: Core modules for companies to manage employee records, roles, and permissions, as well as customer profiles and interactions within their portal.
- Job & Attendance Tracking: Allow companies to create, assign, and track jobs and projects, alongside employee attendance logging with check-in and check-out functionality.
- Financial & Document Management: Enable companies to track earnings, view financial overviews, manage invoices, store critical documents, and record bank transactions.
- AI-powered Reporting Tool: Leverage generative AI as a tool to create dynamic summaries and insights from company operational data, such as employee performance, financial trends, and module utilization.

## Style Guidelines:

- The visual theme is a professional, modern dark mode. The primary brand accent color for interactive elements and highlights is a vibrant orange, specified as #F97316. A secondary, slightly darker orange (#EA580C) will be used for supporting elements and active states. The main background color for the application will be a deep charcoal black (#0F0F0F), with elevated surface elements (cards, panels) appearing as a slightly lighter dark grey (#1A1A1A). All text will be rendered in crisp white (#FFFFFF) for maximum contrast and readability on dark backgrounds.
- The application will primarily use 'Inter', a grotesque sans-serif font. Its modern, objective, and neutral aesthetic ensures excellent readability for both headlines and body text, crucial for data-heavy administrative dashboards.
- Utilize a consistent set of clean, modern line icons that complement the dark theme and sans-serif typography. Icons should be clear and immediately recognizable to enhance user navigation and comprehension within the various modules.
- The application features a responsive web UI with a prominent sidebar navigation for primary modules and a persistent top header displaying the user profile and essential global actions. Dashboards will be composed of cards and tables for quick overviews, supported by forms for data input and detail pages for granular information.
- Implement subtle and purposeful animations for state changes, loading indicators, and user interactions to provide a smooth and engaging user experience without distracting from the data-focused nature of the application.
